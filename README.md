# Validate NIF

A library to validate Portuguese fiscal number, NIF - Número de Identificação Fiscal.

## Installation

```bash
npm install validate-nif --save
```

## Usage

```js
import { validateNIF } from 'validate-nif';

# returns true
validateNIF('123456789');
```
